# **Tirocinio**

## **Avvio del Simulatore**

Per avviare il simulatore, utilizza il seguente comando:

```bash
python ./main.py
```
## **Input**
Per modificare le specie iniziali, aprire `species.txt`
Per modificare i parametri, aprire `parameters.txt`

## **Output**
L'output del programma è su terminale
